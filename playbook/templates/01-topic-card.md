# 测评选题卡

## 1. 一句话商品

- 测评名称：
- 给谁做：
- 用户现在正在问什么：
- 用户完成后想得到什么答案：
- 一句话承诺：完成这套测评，你会知道 ________。

## 2. 需求证据

记录内容平台上至少 10 条相关内容，不要只写自己的猜测。

| 链接/来源 | 标题或主题 | 互动信号 | 评论区用户原话 | 可以借鉴什么 |
|---|---|---:|---|---|
|  |  |  |  |  |

## 3. 购买理由

- 用户为什么不能只看一篇免费内容？
- 他为什么愿意现在就测？
- 结果是否足够具体，能让用户觉得“这说的是我”？
- 结果是否适合截图、收藏或分享？

## 4. 结果原型

| 原型名 | 一句话定义 | 与其他结果的边界 | 用户最想看到的解释 |
|---|---|---|---|
|  |  |  |  |

## 5. 最小完成判定

- [ ] 目标人群明确
- [ ] 至少有 10 条外部需求证据
- [ ] 一句话承诺具体
- [ ] 至少 3 个差异清晰的结果
- [ ] 能用一张封面表达“测完得到什么”
