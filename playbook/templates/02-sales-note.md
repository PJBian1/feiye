# 平台销售内容模板

## 标题

从下面结构中选一个，不虚构数据或身份：

- 测完才发现，我一直把 ________ 当成了 ________
- 如果你也经常 ________，来看看你是哪一种 ________
- 我把 ________ 做成了一套测评
- 你以为自己是 ________，结果可能更像 ________

## 正文结构

### 1. 具体场景

写一个目标用户熟悉的瞬间，不要先讲产品功能。

> 示例：每次进入一个新群体，你会先观察谁最安静，还是先找到最热闹的人？

### 2. 未被说清的问题

说明这个场景背后藏着什么身份、关系或选择问题。

### 3. 测评做了什么

- 题目数量：
- 结果数量：
- 大约需要多久：
- 用户最后会得到什么：

### 4. 展示一个结果切片

展示结果的名字、特征或视觉，但不要把所有内容一次性放完。

### 5. 清楚的行动指令

说明商品入口在哪里、购买后会收到什么，以及这是娱乐与自我表达内容而非专业诊断。

## 发布前检查

- [ ] 没有承诺“绝对准确”或保证人生结果
- [ ] 没有伪造销量、评价、权威身份或文学引文
- [ ] 封面能在三秒内看懂测什么
- [ ] 正文明确写了买家得到的是测评链接
- [ ] 商品说明和实际交付一致
